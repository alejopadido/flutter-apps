# Expense Tracker App
## What is it for?
Keeping track of your expenses
## Features
- Expense categories
- Recording each expense by date
- Deleting expenses
- Chart comparing expenses in different categories
## Flutter Knowledge Applied
- App Theming
- Enums
- Modal Sheet
- Scaffold Messenger
- SnackBar
- Getters
- ListView
- Storing user input with TextEditingController
- Async functions
- Show Dialogs
